# Scripts

Utility scripts for development and maintenance.

## Files

- **run_tests.py** - Test runner with nice output formatting
  - Usage: `python scripts/run_tests.py [test_path]`
  - Shows each test with OK/FAIL status

- **send_template_emails.py** - Email template testing script
  - Usage: `python scripts/send_template_emails.py`
  - Sends test emails to verify email templates
